# Refactor math: разведка

Пользовательский запрос:
- Навести порядок в математике проекта.
- `src/core` — сама математика.
- `src/nucleus` — реализация укладки памяти с помощью новой математики, извлекаются веса из моделей.
- Документация по математике лежит в `Universe`.

Ограничения этапа 2:
- Только исследование, без изменения проектных файлов.
- Нужно найти все места, требующие изменений.
- Обязательно найти и прочитать сопутствующие тесты.
- Нужно подготовить список файлов: код + тесты + документация, которые нужно изменить или учитывать.

Ожидаемый артефакт от research-агента:
- `research-report.md` внутри этой папки.
- В отчете: обзор текущей структуры, проблемы строгости/красоты математики, найденные тесты, список файлов-кандидатов для изменения, риски.
