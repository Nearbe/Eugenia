# REVIEW_SCOPE.md

## Review Target
Alignment of mathematical implementation (`src/core`, `src/nucleus`) with theoretical foundations (`Universe/architecture`, `Universe/Essentials`, `Universe/analytic`).

## In Scope
- **`src/core/*`**: Implementation of U-algebra, calculus, and fractal algorithms.
- **`src/nucleus/*`**: High-level systems (knowledge maps, pattern synthesis) using mathematical primitives.
- **`Universe/architecture/*` & `Universe/Essentials/*`**: Core theoretical axioms and design principles.
- **`Universe/analytic/*`**: Research findings justifying implementation choices.

## Out of Scope
- Peripheral modules (`src/utils`, `src/data`, `src/extractors`).
- Implementation details of rendering/plotting (`src/renderers`).
- Configuration models (`src/models`).
- The expansive reference library (`Universe/Everything/`).

## Primary Goal (Provisional)
Verify that the implementation strictly adheres to the defined mathematical and architectural principles.
